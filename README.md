## 🚀 Kurulum

```bash
# Projeyi klonla
git clone https://github.com/kullaniciAdi/proje-adi.git

# Dizine gir
cd proje-adi

# Gerekli bağımlılıkları yükle
npm install  # veya composer install

# Projeyi başlat
npm run dev  # veya php artisan serve
